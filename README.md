# SIVES_Enterprise
Sistema de Vencimientos de Seguros (SIVES)

Descripción

    SIVES es una plataforma web diseñada para optimizar la gestión de vencimientos de seguros. Facilita el registro, monitoreo y notificación de fechas de vencimiento de pólizas, permitiendo una administración eficiente y minimizando riesgos asociados a vencimientos no controlados.

Características Principales

    Registro y administración centralizada de pólizas de seguro.

    Alertas automáticas para vencimientos próximos.

    Selectores dinámicos para agilizar la selección de agencias y seguros.

    Panel de control intuitivo para visualizar el estado de las pólizas.

    Generación y exportación de informes de vencimientos.

Seguridad avanzada con control de acceso por roles.

    Tecnologías Utilizadas

    Backend: PHP con framework Laravel.

    Frontend: HTML, CSS, JavaScript (incluyendo jQuery).

    Base de Datos: MySQL.

    Servidor: Apache o Nginx.

Uso

    Acceder a la plataforma desde el navegador en http://localhost:8000.

    Iniciar sesión con las credenciales proporcionadas o un usuario de prueba generado.

    Registrar nuevas pólizas y configurar alertas de vencimiento para un seguimiento eficiente.
